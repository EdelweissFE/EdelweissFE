Outputmanagers

Outputmanagers are used for monitoring and exporting results to variuous formats.

Example: testfiles/OutputManagers/test.inp

*section, name=section1, thickness=1.0, material=modleon, type=plane
all

*job, name=cps4job, domain=2d, solver=NIST

*modelGenerator, generator=planeRectQuad, name=gen
x0=0, l=50
y0=0, h=100
elType=CPS4
nX=25
nY=50

*fieldOutput,
name=Displacement, elSet=all, field=displacement, result=U,

ensight - Export to Ensight Gold format

Output manager for Ensight exports. If loaded, it automatically exports all elSets as Ensight parts. For each part, perNode and perElement results can be exported, which are imported from fieldOutputs.

ATTENTION:

fieldOutputs for perNode results must be defined on elSets instead of a nodeSet.

{ 'configuration': 'overwrite=y|n|yes|no|True|False|',
  'create': 'perNode|perElement',
  'fieldOutput': 'name of the result, defined on an elSet (also for perNode '
                 'results)',
  'intermediateSaveInterval': 'step option in category "Ensight": save .case '
                              'file every N increments',
  'name': '(optional), standard = fieldOutputs name'}

meshplot - Export to matplotlib

@author: Magdalena

Module meshplot divided into classes:
  • Triangulation:

    creates triangles out of rectangles in order to use matplotlib plotting options

  • Plotter:

    creates figures, axes, grid, labels

  • Outputmanager:

    creats the plotting specific for the defined keyword lines

{ 'axSpec': 'axis specification according to matplotlib syntax, (default=111)',
  'create=meshOnly': 'plot the mesh only',
  'create=perElement': 'result per element is plotted in a meshplot',
  'create=perNode': 'result per node is plotted in a meshplot',
  'create=xyData': '2D Plot of results',
  'figure': 'figure number, (default=1)'}

Creating finite element meshes

Relevant module fe.utils.abqmodelconstructor

The default way to create finite element meshes is using the keywords

  • *node

  • *element

  • *nset

  • *elset

  • *surface

employing an Abaqus-like syntax.

Example: testfiles/LinearElasticIsotropic/test.inp
*material, name=LinearElastic, id=myMaterial
**Isotropic
**E   | nu |
1.8e4, 0.22

*section, name=section1, material=myMaterial, type=solid
all

*node
0, 0, 0, 0
1, 0.100, 0, 0
2, 0.100, 0.100, 0
3, 0, 0.100, 0
4, 0, 0, 0.100
5, 0.100, 0, 0.100
6, 0.100, 0.100, 0.100
7, 0, 0.100, 0.100

*nSet, nSet=bottom
0,1,2,3
*nSet, nSet=top
4,5,6,7
*nSet, nSet=back
0,3,4,7
*nSet, nSet=front
1,2,5,6
*nSet, nSet=left
0,1,4,5
*nSet, nSet=right
2,3,6,7
*nSet, nSet=bottomleft
1
*nSet, nSet=bottomback
0


*element, Type=C3D8
1, 0,1,2,3,4,5,6,7

*elSet, elSet=all
1

*surface, type=element, name=surfaceTop
**elset, faceID
all, 2
*surface, type=element, name=surfaceFront
**elset, faceID
all, 3
*surface, type=element, name=surfaceRight
**elset, faceID
all, 4
*surface, type=element, name=surfaceBack
**elset, faceID
all, 5
*surface, type=element, name=surfaceLeft
all, 6

*job, name=mySingleElementJob, domain=3d, solver=NIST

*step, jobName=mySingleElementJob, maxInc=1e0, minInc=1e0, maxNumInc=100, maxIter=25, stepLength=1
dirichlet, name=1, nSet=left, field=displacement, 2=0
dirichlet, name=2, nSet=bottomback, field=displacement, 1=0,3=0
dirichlet, name=3, nSet=bottomleft, field=displacement, 3=0

*step, jobName=mySingleElementJob, maxInc=1e0, minInc=1e0, maxNumInc=10, maxIter=25, stepLength=1
dirichlet, name=right, nSet=right, field=displacement, 2=-0.01

*fieldOutput,
elSet=all, field=displacement, result=U, name=displacement
elSet=all, result=strain, name=strain, quadraturePoint=0:8, f(x)='np.mean(x,axis=1)'
elSet=all, result=stress, name=stress, quadraturePoint=0:8, f(x)='np.mean(x,axis=1)'
name=sig22, elSet=all, result=stress, quadraturePoint=0:8, f(x)='-np.mean(x[:,:,1],axis=1)', saveHistory=True
name=eps22, elSet=all, result=strain, quadraturePoint=0:8, f(x)='-np.mean(x[:,:,1],axis=1)', saveHistory=True, export=eps22

*output, type=ensight, jobName=mySingleElementJob, name=myEnsightOutput
create=perNode, fieldOutput=displacement
create=perElement, fieldOutput=stress
create=perElement, fieldOutput=strain
configuration, overwrite=yes

*output, type=meshPlot, jobName=mySingleElementJob
figure=1, axSpec=111, c=red, create=xyData, x=eps22, y=sig22

*output, type=monitor, jobName=mySingleElementJob
fieldOutput=sig22
